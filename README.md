# 🎴 셜록 고스톱 정산소 시스템

우리 동아리만의 독특한 **"파산자 2달러 분배 법칙"**을 데이터 유실 없이 실시간으로 계산하고 누적하는 전용 웹 애플리케이션입니다.

## 🚀 아키텍처 특장점
- **Vanilla JS + Module 시스템:** 빌드나 웹팩 설치 과정 없이 브라우저 가볍게 즉시 실행.
- **Tailwind CSS UI:** 모바일 화면(스마트폰)과 태블릿, PC 전체를 완벽 대응하는 반응형 그리드 시스템.
- **Firebase Firestore Transaction:** 동시 정산 승인 요청이 일어날 때 연산 오류 및 돈 복사 현상을 100% 방지하는 데이터 무결성 설계.

## ⚙️ 초기 설정 방법
1. 본 프로젝트 폴더를 다운로드받거나 Git Clone합니다.
2. `app.js` 파일의 4번째 줄에 본인의 Firebase 웹 자산 설정을 붙여넣습니다.
3. 파일들을 Public GitHub Repository에 푸시한 뒤 **Settings -> Pages**를 켜면 즉시 24시간 실시간 호스팅 서버가 활성화됩니다!
